# ⚡ NOVA Browser (Work in Progress)

NOVA is a **custom futuristic desktop web browser** built using **Electron and Chromium** with a **neon cyber UI**, **custom start page**, **dynamic tabs**, and **personalized welcome system**.

🚧 **Status:** Actively under development  
👨‍💻 **Developer:** Arush  
🎯 **Goal:** Build a fully functional modern browser with a cyberpunk UI and advanced features.

---

## ✅ Current Features (Implemented)

- ✅ Chromium-based web rendering
- ✅ Futuristic neon glass UI
- ✅ Transparent top bar & sidebar
- ✅ Custom start / home page
- ✅ Neon search bar
- ✅ Quick access website tiles
- ✅ “+ Add Website” button with Local Storage
- ✅ Persistent custom shortcuts
- ✅ Dynamic Tabs UI
- ✅ New Tab button
- ✅ Personalized Welcome Message: **“Welcome Arush”**
- ✅ Navigation controls (Back, Forward, Reload)
- ✅ Download handling via Electron
